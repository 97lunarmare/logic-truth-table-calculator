Propositional Logic Truth Table Calculator

Enter propositional logic into the calculator to see its truth table.

You can use and, or, not, then, equals, xor as well as symbols like -> for then etc. Separate pieces of logic using a comma to compare their truth tables, e.g `p then q, !p or q. You can also use T or true to specify true and false values.
